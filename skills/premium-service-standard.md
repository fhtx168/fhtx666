### 2️⃣ 内容质量（要精品）

**搜索摘要提炼标准**：

| 维度 | 要求 |
|------|------|
| **标题** | 准确反映核心观点，不标题党 |
| **摘要** | 提炼核心逻辑，不是简单复制 |
| **分类** | 精准归类（大盘/板块/个股/政策/风控） |
| **关键词** | 提取 3-5 个核心关键词 |
| **价值评级** | 高/中/低，优先保留高价值 |

**极品标准**：

### 3️⃣ 周报质量（要有深度）

**六大类内容标准**：

| 类别 | 内容要求 | 极品标准 |
|------|----------|----------|
| **大盘趋势** | 目标位 + 关键变量 + 时间窗口 | 有逻辑、有数据、可验证 |
| **板块逻辑** | 主线 + 轮动 + 产业链 | 说清为什么、能持续多久 |
| **个股机会** | 龙头 + 潜力股 + 目标价 | 有逻辑支撑，不是瞎猜 |
| **仓位节奏** | 建议仓位 + 加减仓时机 | 明确、可执行 |
| **风控避雷** | 风险点 + 避雷案例 | 具体、有警示价值 |
| **后市预判** | 关键时点 + 事件 + 变量 | 可跟踪、可验证 |

**极品周报特征**：
- ✅ 不是信息堆砌，而是有洞察
- ✅ 不是模棱两可，而是有观点
- ✅ 不是事后诸葛亮，而是事前预判
- ✅ 不是泛泛而谈，而是可执行

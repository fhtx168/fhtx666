# 渠道适配说明

## 飞书渠道

自动启用优化模式：

### 调用前
```
📹 正在调用 xxx 技能...
⏳ 当前工具执行时间较长，请耐心等候...
```

### 调用后
根据内容类型分别发送：
1. **文本内容** → 普通消息
2. **图片** → 每张单独发送
3. **视频** → 卡片消息 + 按钮

### 错误处理
显示红色错误卡片：
```
❌ xxx 执行出错
错误信息：xxx
```

## 其他渠道

输出标准 JSON：
```json
{
  "text": "...",
  "images": [...],
  "videos": [...],
  "documents": []
}
```

## 自动识别

优先顺序：
1. 命令行 `--channel` 参数
2. 环境变量 `OPENCLAW_CHANNEL`
3. 默认 `web`

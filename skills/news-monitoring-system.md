## 五、资讯解读与决策融合

### 5.1 解读框架

**每条重大资讯需分析**：

```
1. 事件本身：发生了什么？
2. 政策背景：为什么现在出台？
3. 影响范围：哪些行业/公司受益/受损？
4. 持仓关联：对持仓股有什么影响？
5. 操作建议：买入/卖出/观望？
6. 时间窗口：短期/中期/长期影响？
```

### 6.1 资讯抓取脚本

```python
# skills/news-monitor.py
# 监控渠道资讯，关键词过滤，分级推送

import feedparser
import requests
import json
from datetime import datetime

# 监控渠道配置
CHANNELS = {
    '新华社': 'http://www.xinhuanet.com/politics/rss.xml',
    '财联社': 'https://www.cls.cn/rss',
    '华尔街见闻': 'https://wallstreetcn.com/rss',
    # ...
}

# 关键词配置
KEYWORDS_P0 = ['政治局会议', '降准', '降息', '贸易战', '涨停', '业绩预增']
KEYWORDS_P1 = ['产业政策', '行业监管', '重组', '收购']
KEYWORDS_HOLDINGS = ['华工科技', '比亚迪', '西藏珠峰', ...]

def fetch_news(channel, url):
    """抓取资讯"""
    feed = feedparser.parse(url)
    return feed.entries

def filter_by_keywords(entries, keywords):
    """关键词过滤"""
    filtered = []
    for entry in entries:
        for keyword in keywords:
            if keyword in entry.title or keyword in entry.summary:
                filtered.append({
                    'channel': channel,
                    'title': entry.title,
                    'summary': entry.summary,
                    'link': entry.link,
                    'published': entry.published,
                    'keyword': keyword
                })
                break
    return filtered

def classify_priority(news):
    """分级"""
    if any(k in news['title'] for k in KEYWORDS_P0):
        return 'P0'
    elif any(k in news['title'] for k in KEYWORDS_P1):
        return 'P1'
    else:
        return 'P2'

def send_alert(news, priority):
    """推送"""
    # 调用微信推送 API
    pass

# 主循环
def monitor():
    for channel, url in CHANNELS.items():
        entries = fetch_news(channel, url)
        filtered = filter_by_keywords(entries, KEYWORDS_P0 + KEYWORDS_P1)
        for news in filtered:
            priority = classify_priority(news)
            send_alert(news, priority)
```

# 财经资讯舆情情绪分析

对个股/持仓的财经资讯进行情绪分析，辅助判断市场情绪走向。

## 使用场景

1. **单只股票情绪**：给定股票名称/代码，分析其近期舆情
2. **持仓组合情绪**：扫描全部或部分持仓，给出整体情绪画像
3. **重点标的监控**：指定持仓股名单，快速筛查异常舆情

## 分析流程

### Step 1：搜索资讯

使用搜索工具搜索目标股票近 7-14 天的资讯：
- 来源：360搜索、必应
- 关键词：`{股票名} 利好/利空/公告/研报/政策`
- 每只股最多取 10 条资讯

### Step 2：情绪分析

使用 DeepSeek 或通义千问，对每条资讯判断：

```
情绪分类：利好 / 利空 / 中性
情绪强度：强 / 中 / 弱
资讯类别：公告 / 政策 / 业绩 / 行业 / 舆情 / 资金面
```

### Step 3：综合输出

```
## {股票名} 舆情情绪报告
📅 分析日期 | 📊 搜索条数

### 情绪评分
综合评级：看多 ★★★☆☆ / 中性 / 看空
利好条数：X | 利空条数：Y | 中性条数：Z

### 核心利好
1. ...
2. ...

### 核心利空（风险）
1. ...
2. ...

### 情绪趋势
较前一周：情绪升温/持平/降温
```

## 注意事项

- 情绪分析为主观判断，仅供参考
- 优先关注官方公告和权威媒体报道
- 忽略自媒体/股吧等低质量信息源
- 如果某只股票近7天无重要资讯，如实告知
- 持仓股分析时，结合持仓盈亏状态给出建议

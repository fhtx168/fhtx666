# 技术指标计算与图表分析

计算股票技术指标并辅助趋势判断。**仅按需执行，计算完成后立即释放，零资源占用。**

## 支持的指标

| 指标 | 周期 | 用途 |
|------|------|------|
| MA | 5/10/20/60/120/250日 | 均线系统，趋势方向 |
| EMA | 12/26日 | 指数均线，灵敏度更高 |
| MACD | DIF/DEA/HIST | 趋势动能与背离 |
| RSI | 6/12/24日 | 超买超卖（>70超买, <30超卖） |
| KDJ | 9/3/3 | 短中线买卖信号 |
| BOLL | 20日 | 布林带，波动区间 |
| VOL | 5/10日均量 | 量能变化 |

## 数据来源

Tushare Pro `daily` 接口，获取近 250 日 K 线数据。

## 计算方式

**纯 Python 实现**，不依赖 numpy/pandas 等重型库。计算公式见 `references/formulas.md`。

## 使用流程

### Step 1：拉取数据
通过 Tushare Pro API 获取日线数据（开高低收量）。

### Step 2：计算指标
按用户需求计算 1-N 个指标，每次最多同时分析 5 只股票。

### Step 3：趋势判断
基于指标组合给出简洁判断：
- 趋势方向：上升/震荡/下降
- 关键位置：支撑位/压力位
- 信号：金叉/死叉/背离

### Step 4：图表（可选，仅用户要求时执行）
如用户要求生成 K 线图，使用轻量方式生成图片，执行完即释放。

## ⚠️ 注意事项

- 技术分析基于历史数据，**不预测未来**
- 仅作为中长线辅助参考，不替代基本面分析
- 单次最多分析 5 只股票，避免批量请求超时
- 图表生成仅在有明确需求时触发
- 计算完成后立即释放，不驻留任何进程

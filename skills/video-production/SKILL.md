## 🔧 核心代码片段

### 音频生成
```python
import edge_tts
import asyncio

async def generate_audio(text, output_file):
    communicate = edge_tts.Communicate(text, "zh-CN-XiaoxiaoNeural")
    await communicate.save(output_file)
```

### 视频合成
```python
from moviepy import *

# 加载音频
audio = AudioFileClip("audio.mp3")
duration = audio.duration

# 创建图片剪辑
image_clip = ImageClip("background.jpg").with_duration(duration)
image_clip = image_clip.with_effects([vfx.CrossFadeIn(0.5), vfx.CrossFadeOut(0.5)])

# 合成视频
final = image_clip.with_audio(audio)
final.write_videofile("output.mp4", fps=24)
```

## ⚠️ 常见问题与解决

### 问题 1：中文路径乱码
**现象**：PowerShell/Python GBK 编码错误  
**解决**：移除 emoji 和中文字符，使用英文路径

### 问题 2：Unsplash API 关闭
**现象**：无法获取真实风景图  
**解决**：本地图片库或渐变背景替代

### 问题 3：渲染速度慢
**现象**：MoviePy 逐帧渲染耗时  
**解决**：接受现实，质量优先（v5 渲染 1 小时 26 分钟）

### 问题 4：音画不同步
**现象**：音频时长与画面不匹配  
**解决**：`with_duration(audio.duration)` 强制同步

## 💡 经验教训

### 成功经验
1. **语音选择关键** — 女声晓晓比男声云健更适合散文
2. **图片匹配重要** — 真实风景图比渐变背景更有代入感
3. **版本迭代必要** — v1→v5 逐步优化，质量提升明显
4. **耐心渲染值得** — 慢工出细活，质量优先

### 待改进
1. **渲染速度** — 探索 FFmpeg 加速
2. **图片源** — 建立本地风景图库
3. **自动化** — 一键生成流程
4. **字幕** — 自动同步字幕

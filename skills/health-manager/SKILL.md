## 核心能力

### 1. 体检报告解析
**当前能力**: ✅ 基础文本提取  
**限制**: ❌ 扫描版 PDF 需要 OCR  
**解决方案**: 
- 已安装 `pdf2image` + `pytesseract`
- 需要配置 Tesseract OCR 引擎
- 或使用在线 OCR 服务（百度 OCR、腾讯 OCR）

**支持格式**:
- ✅ 文本型 PDF
- ⏳ 扫描版 PDF（需要 OCR）
- ✅ Excel/CSV 体检数据
- ✅ 手动输入的指标数据

### 2. 健康指标分析
**能力范围**:
- ✅ 血常规指标分析
- ✅ 生化指标分析（肝功、肾功、血脂、血糖）
- ✅ 肿瘤标志物解读
- ✅ 影像学检查结果分析
- ✅ 历年数据对比

**分析方法**:
1. 与正常参考范围对比
2. 异常程度分级（轻度/中度/重度）
3. 指标关联性分析
4. 趋势预测（如有历年数据）

### 3. 健康风险评估
**评估维度**:
- 代谢性疾病风险（糖尿病、高血脂、痛风）
- 心血管疾病风险
- 肝脏疾病风险
- 肾脏疾病风险
- 肿瘤风险筛查

**评估方法**:
- 基于临床指南和风险评分模型
- 多指标联合评估
- 生活方式因素考量

### 4. 健康干预建议
**建议类型**:
- 饮食调整方案
- 运动处方
- 睡眠改善建议
- 压力管理方案
- 补充剂建议
- 就医指导

**个性化程度**:
- 基于体检结果
- 考虑年龄、性别、职业
- 结合既往病史
- 可执行性强

## 当前限制与补足方案

### 限制 1: 扫描版 PDF 解析
**问题**: 体检报告是扫描版，文本提取失败（仅 33 字符）

**解决方案**（三选一）:

#### 方案 A: 本地 OCR（推荐）
**步骤**:
1. 安装 Tesseract OCR 引擎
   ```powershell
   # 下载 https://github.com/UB-Mannheim/tesseract/wiki
   # 安装 tesseract-ocr-w64-setup-5.x.x.exe
   ```
2. 配置环境变量
   ```powershell
   $env:TESSERACT_PATH = "C:\Program Files\Tesseract-OCR\tesseract.exe"
   ```
3. 执行 OCR 提取
   ```python
   from pdf2image import convert_from_path
   import pytesseract
   
   images = convert_from_path('health_checkup_202603.pdf')
   text = pytesseract.image_to_string(images[0], lang='chi_sim+eng')
   ```

**优点**: 本地处理，隐私保护好  
**缺点**: 需要安装额外软件，中文识别率约 85-95%  
**时间**: 30 分钟配置完成

#### 方案 B: 在线 OCR API
**推荐服务**:
- **百度 OCR**: https://cloud.baidu.com/product/ocr
  - 免费额度：500 次/月
  - 中文识别率：98%+
  - 支持表格识别
  
- **腾讯 OCR**: https://cloud.tencent.com/product/ocr
  - 免费额度：1000 次/月
  - 中文识别率：98%+
  - 支持医疗报告专用识别

**集成方式**:
```python
import requests

def ocr_medical_report(pdf_path):
    # 转换为图片后调用 API
    # 返回结构化体检数据
    pass
```

**优点**: 识别率高，支持表格结构化  
**缺点**: 需要网络，隐私考虑  
**时间**: 1 小时集成完成

#### 方案 C: 手动输入关键指标（临时方案）
**方式**: 您提供关键异常指标，我立即分析

**模板**:
```
【基本信息】
年龄：XX 岁
性别：男/女

【异常指标】
1. 指标名称：数值（参考范围）- 偏高/偏低
2. ...

【检查结论】
1. ...
2. ...
```

**优点**: 立即开始分析  
**缺点**: 需要手动输入  
**时间**: 5 分钟

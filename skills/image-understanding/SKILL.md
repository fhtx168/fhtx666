## 使用方法

### 1. OCR 文字识别（推荐）

**适用**: 截图、文档、资讯图片、包含文字的图片

```
识别这张图片中的文字
读取截图中的内容
OCR: path/to/image.png
```

**工具选择**:
- **360-ocr.js**: 免费，适合日常使用
- **dashscope-ocr.js**: 阿里云，精度更高

**示例**:
```bash
# 使用 360 OCR
C:\Users\Admin\opcclawai\project\skills\360-ocr\scripts\360-ocr.js C:\path\to\image.jpg

# 使用阿里云 OCR
C:\Users\Admin\opcclawai\project\skills\ali-ocr\scripts\dashscope-ocr.js C:\path\to\image.jpg
```

---

### 2. 图片内容理解

**适用**: 场景识别、物体检测、人物分析、图片描述

**方法**: 使用 browser 工具加载图片，然后用 AI 分析

**示例**:
```
描述这张图片的内容
这张图里有什么？
分析这张图片的场景和物体
```

**执行流程**:
1. 用 `read` 工具读取图片
2. 用 AI 模型分析图片内容
3. 输出结构化描述

---

### 3. 图表数据提取（🟡 开发中）

**适用**: K 线图、柱状图、折线图、饼图、数据表格

**计划功能**:
- 提取图表中的数值数据
- 识别趋势和关键点
- 转换为 Excel/CSV 格式

**临时方案**: 手动描述图表，AI 辅助整理

---

### 4. 视频字幕提取

**适用**: 本地视频文件、在线视频链接

**方法**: 使用 `audio_to_text` skill

**示例**:
```
提取这个视频的字幕
把视频内容转成文字
下载并转写这个视频
```

**执行流程**:
1. 下载视频（如需要）
2. 提取音频
3. 语音识别转文字
4. 生成 SRT 字幕文件

## 限制与注意事项

1. **图片大小**: 单张图片 ≤ 10MB
2. **格式支持**: JPG, PNG, WEBP, GIF, BMP
3. **语言**: 中文 + 英文混合识别效果最佳
4. **图表提取**: 复杂图表可能需要人工校验
5. **视频时长**: 建议 ≤ 30 分钟，长视频处理时间较长

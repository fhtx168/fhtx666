# 投资组合简易风控

对持仓组合进行风险指标计算和预警，辅助中长线投资决策。

## 风控维度

### 单票风险
- **涨跌幅预警**：单日涨跌幅超过 ±5% 标记
- **回撤预警**：从阶段高点回撤超过 10% 标记
- **持仓盈亏**：浮盈超过 50% → 高低切候选；浮亏超过 20% → 基本面预警

### 组合风险
- **仓位集中度**：单票占总仓位比例，超过 10% 标记
- **行业集中度**：单一行业占比超过 40% 标记
- **行业分布**：按科技/能源资源/生物制药军工分类统计

## 数据来源

- **持仓信息**：从 `portfolio/holdings.md` 读取
- **实时行情**：通过 Tushare Pro `daily` 接口获取
- **阶段高点**：通过 Tushare `daily` 获取近60日行情计算

## 输出格式

```
## 📊 风控报告 {日期}

### ⚠️ 预警个股
| 股票 | 预警类型 | 当前值 | 阈值 | 建议 |
|------|----------|--------|------|------|

### 组合健康度
- 总仓位集中度 TOP3: ...
- 行业分布: 科技 X% | 能源 X% | 医药军工 X%
- 整体浮盈/浮亏: ...

### 💡 操作建议
- 高低切候选（涨幅过高）: ...
- 需关注基本面（跌幅过大）: ...
```

## 阈值配置（可调整）

| 指标 | 默认阈值 |
|------|----------|
| 单日涨跌幅 | ±5% |
| 阶段回撤 | >10% |
| 浮盈（高低切候选） | >50% |
| 浮亏（基本面预警） | >20% |
| 单票集中度 | >10% |
| 行业集中度 | >40% |

## 注意事项

- 风控信号仅为提示，不构成操作建议
- 中长线策略容忍度高于短线，阈值偏宽松
- 预警不等同于止损信号

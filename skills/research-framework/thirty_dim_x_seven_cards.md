## 🔄 选股流程（五步法）

### 第一步：三十五维全面分析（100 分）

```
执行完整的三十五维 v4.0 评估，输出：
- 五层穿透评分（表层信号 8% + 基本面 28% + 护城河 22% + 资金博弈 18% + 决策行动 24%）
- 三十五个维度详细评分（含新增 31-35 维度）
- 综合星级评级（1-5 星）
```

**工具**: `skills/research-framework/framework-v4.0.md` + `memory/2026-05-23.md`（三十五维 v4.0 应用实例）

### 新增 5 维度（31-35）详解

| 维度 | 名称 | 核心逻辑 | 七张底牌映射 |
|------|------|----------|--------------|
| **31** | Token 经济性 | 算力变现能力、Token 处理量/月 | 第 1 张（AI 算力端） |
| **32** | 技术范式 | AI 核心度、是否生成式 AI→物理 AI | 第 1/2/3 张（芯片/光模块/封装） |
| **33** | 关键材料 | 稀缺性、锗/铀/银/锡/钼等资源 | 第 7 张（有色金属） |
| **34** | 催化窗口 | 6-9 月密集期（英伟达财报/长鑫 IPO/H1 验证） | 全部七张底牌 |
| **35** | 大厂背书 | 大基金/华为哈勃/英伟达认证 | 第 2/3/5 张（国产芯片/供应链） |

**评分权重**: 新增 5 维度占总评分 15%（每维 3%）

### 第二步：七张底牌契合度评估（100 分）

```
评估标的与七张底牌的契合程度：
- 完美契合（90-100 分）：直接对应七张底牌核心逻辑
- 高度契合（70-89 分）：间接支撑七张底牌逻辑
- 一般契合（50-69 分）：部分相关但非核心
- 不契合（<50 分）：与七张底牌逻辑冲突
```

**工具**: `yrt_monitoring/yrt_complete_knowledge_system.md`

### 第三步：双重过滤

```
条件 1: 三十维综合评分 ≥ 80 分（4 星以上）
条件 2: 七张底牌契合度 ≥ 70 分（高度契合以上）

同时满足 → S 级标的（重点配置）
仅满足 1 → A 级标的（卫星配置）
均不满足 → 观察或回避
```

### 第四步：催化剂验证

```
检查未来 6-12 个月催化剂：
- 业绩催化（半年报/年报/季报）
- 产品催化（新品发布/产能释放）
- 政策催化（产业政策/补贴）
- 事件催化（IPO 上会/大会/展会）

至少 1 个 P0 级催化剂 → 可立即建仓
仅有 P1/P2 级催化剂 → 等待更好时机
无催化剂 → 暂缓
```

### 第五步：交易计划生成

```
输出完整的交易计划：
- 建仓价格区间
- 目标价格（1 年/3 年）
- 止损价格
- 仓位配置（核心/卫星）
- 加仓/减仓条件
```

## ⚠️ 风险提示

1. **三十维评分滞后性**: 基于历史数据，不能完全预测未来
2. **七张底牌主观性**: 契合度评估有一定主观判断
3. **催化剂不确定性**: 事件可能延期或取消
4. **市场系统性风险**: 大盘暴跌时所有评级失效

**应对策略**:
- 严格执行止损纪律
- 分散配置（单只不超过 15%）
- 定期复核（至少每周一次）
- 关注叶荣添最新观点变化

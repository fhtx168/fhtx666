## 核心升级内容

### 1️⃣ 新增长鑫 IPO 催化评估维度

**背景**：长鑫存储 IPO 是 2026 年半导体产业最大事件，将带动核心供应链企业价值重估。

**新增内容**：
- 设备优先级评估矩阵（刻蚀/薄膜沉积/清洗设备优先级最高）
- 长鑫 IPO 催化评分标准（90-100 分：核心设备供应商直接供货）
- IPO 进度跟踪节点（辅导备案→招股书申报→问询回复→过会审核→发行定价→正式上市）

**应用方法**：
```
长鑫 IPO 催化评分 = 设备优先级评分 × 供应链认证等级 × IPO 进度系数
```

**受益标的**：
- 刻蚀设备：中微公司、北方华创
- 薄膜沉积：拓荆科技、北方华创
- 清洗设备：盛美上海、至纯科技

---

### 2️⃣ 新增叶荣添九剑主线契合度评估

**背景**：叶荣添 2026 年九大投资主线代表未来 3-5 年核心投资方向。

**九剑主线及权重**：
| 主线 | 权重 | 核心逻辑 |
|------|------|---------|
| CCL（覆铜板） | 12% | AI 服务器需求爆发，高频高速板材紧缺 |
| 锡 | 10% | AI 算力 + 光伏双驱动，供给刚性 |
| 银 | 10% | 光伏银浆 + 电子焊料，工业金属之王 |
| 光模块 | 15% | AI 算力核心，800G/1.6T 迭代加速 |
| 电力 | 12% | AI 数据中心耗电激增，电力紧缺 |
| 封装 | 12% | 先进封装是摩尔定律延续关键 |
| 钼 | 8% | 军工 + 特钢双驱动，供给受限 |
| 液冷 | 11% | AI 服务器散热刚需，渗透率快速提升 |
| PCB | 10% | AI 服务器 + 汽车电子双轮驱动 |

**评分标准**：
- 🔥 90-100 分：九剑核心标的，直接受益
- 🟢 70-89 分：主线相关，间接受益
- 🟡 50-69 分：部分相关，概念沾边
- 🔴 <50 分：无关联或负相关

**计算公式**：
```
主线契合度得分 = Σ(各主线匹配度 × 主线权重)
```

### 3️⃣ 新增半导体设备国产化专项评分

**背景**：半导体设备国产化是国家战略，长鑫存储 + 长江存储双轮驱动。

**国产化率现状（2026 年）**：
| 设备类型 | 国产化率 | 目标（2027） | 提升空间 |
|---------|---------|------------|---------|
| 刻蚀设备 | ~35% | 50% | ⭐⭐⭐⭐⭐ |
| 薄膜沉积 | ~25% | 40% | ⭐⭐⭐⭐⭐ |
| 清洗设备 | ~40% | 60% | ⭐⭐⭐⭐ |
| 量测设备 | ~15% | 30% | ⭐⭐⭐⭐⭐ |
| CMP 设备 | ~30% | 50% | ⭐⭐⭐⭐ |

**供应链认证等级**：
- A+ 级：长鑫 + 长江双认证，批量供货（95-100 分）
- A 级：长鑫或长江单认证，批量供货（80-94 分）
- B+ 级：双认证，小批量供货（65-79 分）
- B 级：单认证，送样验证中（50-64 分）
- C 级：潜在供应商，无实质进展（<50 分）

**专项评分维度**：
1. 技术实力（30%）
2. 客户认证（30%）
3. 产能保障（20%）
4. 研发投入（20%）

### 4️⃣ 新增催化剂强度评估

**催化剂类型与权重**：
| 类型 | 权重 | 说明 |
|------|------|------|
| IPO 催化 | 40% | 长鑫 IPO 进度、产业链 IPO |
| 政策催化 | 35% | 产业政策、补贴、税收优惠 |
| 业绩催化 | 25% | 业绩超预期、订单爆发、产能释放 |

**强度分级**：
- 强（90-100 分）：核心事件落地，直接受益
- 中（60-89 分）：事件推进中，预期升温
- 弱（<60 分）：仅概念关联，无实质进展

**计算公式**：
```
催化剂强度得分 = IPO 催化得分×40% + 政策催化得分×35% + 业绩催化得分×25%
```

**应用建议**：
- 强催化（≥80 分）：重点配置，短期有望爆发
- 中催化（60-79 分）：标准配置，等待催化兑现
- 弱催化（<60 分）：低配或观望

## 使用指南

### 适用场景
✅ 三到五年长期投资
✅ A 股 + 港股深度分析
✅ 半导体/硬科技产业链投资
✅ 主线驱动型投资策略

### 不适用场景
❌ 短线投机交易
❌ 纯题材炒作
❌ 缺乏基本面支撑的概念股

### 使用流程
1. **执行三十维分析** - 按照框架逐项评分
2. **评估主线契合度** - 九剑主线 + 七张底牌双重评估
3. **评估催化强度** - IPO/政策/业绩三维催化
4. **输出综合评分** - 加权计算，得出星级评级
5. **制定交易计划** - 根据评分确定仓位和目标价

### 评分速查
| 评分 | 评级 | 建议仓位 |
|------|------|---------|
| 90-100 | ⭐⭐⭐⭐⭐ | 重仓（10-15%） |
| 80-89 | ⭐⭐⭐⭐ | 重点（5-10%） |
| 70-79 | ⭐⭐⭐ | 标准（3-5%） |
| 60-69 | ⭐⭐ | 观察（1-3%） |
| <60 | ⭐ | 回避（0%） |

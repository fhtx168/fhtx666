## 📊 研报分析框架

### 1. 研报统计

| 指标 | 计算方法 | 说明 |
|------|----------|------|
| **研报数量** | 近 30 天/90 天/180 天篇数 | 热度指标 |
| **评级分布** | 买入/增持/中性/减持/卖出占比 | 情绪指标 |
| **目标价均值** | 所有研报目标价平均 | 一致预期 |
| **目标价中位数** | 所有研报目标价中位 | 去极端值 |
| **最高目标价** | 最乐观预期 | 多头观点 |
| **最低目标价** | 最悲观预期 | 空头观点 |
| **上涨空间** | (目标价均值 - 现价)/现价 | 预期涨幅 |

---

## 📋 研报整合 SOP

### 步骤 1：获取研报列表

```powershell
# 东方财富 API
$url = "https://datacenter.eastmoney.com/securities/api/data/v1/get?reportName=RPT_RES_ORG_RPT&filter=(SECUCODE=`"601012.SH`")&pageNumber=1&pageSize=20"
$reports = Invoke-RestMethod -Uri $url
```

### 步骤 2：下载 PDF 研报

```powershell
# 批量下载
foreach ($report in $reports.data) {
    $pdfUrl = $report.PDF_URL
    $outputFile = "./research-pdfs/$($report.ORG_NAME)-$($report.REPORT_TITLE).pdf"
    Invoke-WebRequest -Uri $pdfUrl -OutFile $outputFile
}
```

### 步骤 3：解析 PDF 研报

```bash
# 使用 research-parser 技能
python scripts/parse_research.py --input-dir ./research-pdfs/ --output-dir ./output-json/
```

### 步骤 4：生成研报汇总

```bash
# 生成 Excel 汇总表
python scripts/parse_research.py --input-dir ./research-pdfs/ --excel-output ./research-summary.xlsx
```

## 💡 研报应用建议

### 飞鸿，研报使用原则：

| 原则 | 说明 |
|------|------|
| **参考不盲从** | 研报是参考，不是决策依据 |
| **看一致性预期** | 关注均值，不极端 |
| **跟踪评级变化** | 评级下调要警惕 |
| **精读深度报告** | 深度报告 > 点评报告 |
| **关注头部券商** | 中信/中金/华泰更可靠 |
| **警惕利益冲突** | 承销券商可能偏多 |

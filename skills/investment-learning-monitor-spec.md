## 📋 执行流程

### 步骤 1：核心层监测（每 1 小时）

```
平台：微博、雪球、东方财富
策略：web_search + web_fetch 全文
关键词：叶荣添 + 但斌 + 林园
```

### 步骤 2：优质层监测（每 2 小时）

```
平台：搜狗微信、今日头条、B 站
策略：web_search 摘要
关键词：叶荣添 + 财经大佬
```

### 步骤 3：扩展层监测（每日）

```
平台：抖音、知乎、新浪财经
策略：web_search 摘要
关键词：叶荣添 + 行业观点
```

## 🔧 OpenClaw 工具调用

### web_search 调用示例

```javascript
await web_search({
  query: "叶荣添 微博 最新观点",
  count: 10
})
```

### web_fetch 调用示例（仅核心层）

```javascript
await web_fetch({
  url: "https://weibo.com/1364334665/...",
  extractMode: "markdown"
})
```

## 🎯 内容提炼框架

### 六维提炼

1. **大盘判断** - 趋势/目标位/关键变量
2. **板块逻辑** - 主线/轮动/产业链
3. **个股机会** - 龙头/潜力股/目标价
4. **政策解读** - 宏观/行业/影响
5. **风控要点** - 风险/避雷/仓位
6. **后市预判** - 时点/事件/变量

### 摘要格式

```markdown
# {标题}

> 来源：{平台}
> 时间：{发布时间}
> 链接：{URL}

---

## 核心观点

{搜索摘要/全文提炼}

---

## 分类

{大盘/板块/个股/政策/风控}

---

## 关键词

{关键词 1} | {关键词 2} | {关键词 3}
```

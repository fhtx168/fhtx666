# 券商研报结构化解析

将券商研究报告解析为结构化投资信息，辅助中长线投资决策。

## 输入来源

1. **用户提供的文件**：PDF/DOCX 研报文件
2. **用户提供的 URL**：研报网页链接
3. **按股票搜索**：给定股票代码/名称，搜索并获取最新研报

## 解析流程

### Step 1：获取研报内容

```
# 如果是文件路径 → 用 PDF/DOCX skill 提取文本
# 如果是 URL → 用 web_fetch 抓取内容
# 如果只给股票名 → 搜索最新研报（东方财富/雪球/慧博）
```

### Step 2：结构化提取

使用 DeepSeek 或通义千问大模型，将研报内容解析为以下结构：

```json
{
  "stock_name": "股票名称",
  "stock_code": "股票代码",
  "broker": "券商名称",
  "analyst": "分析师",
  "report_date": "发布日期",
  "rating": "买入/增持/中性/减持",
  "target_price": "目标价(元)",
  "current_price": "当前价(元)",
  "upside": "预期涨幅%",
  "core_thesis": "核心投资逻辑(3-5条要点)",
  "financial_forecast": {
    "revenue_2026": "2026E营收(亿)",
    "revenue_2027": "2027E营收(亿)",
    "net_profit_2026": "2026E净利润(亿)",
    "net_profit_2027": "2027E净利润(亿)",
    "eps_2026": "2026E EPS",
    "eps_2027": "2027E EPS",
    "pe_2026": "2026E PE",
    "pe_2027": "2027E PE"
  },
  "key_catalysts": "关键催化剂/驱动因素",
  "risks": "风险提示(3-5条)",
  "sentiment": "看多/中性/看空"
}
```

### Step 3：输出给用户

以简洁的中文摘要呈现，突出：
- 评级与目标价
- 核心逻辑（要点式）
- 与用户持仓的关联（如果匹配持仓股，标注持仓盈亏状态）
- 风险提示

## 注意事项

- 研报观点仅供参考，不构成投资建议
- 注意区分客观数据和主观判断
- 如果研报质量差或信息不全，如实告知用户
- 对比多家券商观点时，标注一致性和分歧点

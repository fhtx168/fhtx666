## 数据验证机制

### 自动验证
```python
def validate_data(df, required_columns):
    if df is None or len(df) == 0:
        raise ValueError("数据为空")
    
    for col in required_columns:
        if col not in df.columns:
            raise ValueError(f"缺少必要列：{col}")
    
    return True
```

### 人工验证
- 每晚 21:00 推送后检查数据准确性
- 发现异常立即修正并记录到 `memory/heartbeat-issues.md`

## 错误处理

### 网络异常
- 指数退避重试（1s, 2s, 4s）
- 最多重试 3 次
- 仍失败时记录错误并告警

### 数据异常
- 数据为空时标记
- 数据明显错误时（如今日盈亏 +3801 万）标记
- 推送中明确标注"数据缺失"

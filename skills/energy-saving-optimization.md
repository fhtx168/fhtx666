## ✅ 抓取策略（严格执行）

### 只拿搜索摘要

```python
# ✅ 正确：只访问搜索 URL，拿摘要
url = "https://weixin.sogou.com/weixin?type=2&query=叶荣添"
response = requests.get(url, timeout=15)
# 不解析 JS，不 fetch 全文

# ❌ 错误：强行抓取全文
if response.status_code != 200:
    retry()  # 不重试
web_fetch(url)  # 不强行 fetch
```

### 不重试原则

| 状态码 | 处理 |
|--------|------|
| 200 | ✅ 正常处理 |
| 403 | ⚠️ 记录，不重试 |
| 412 | ⚠️ 记录，不重试 |
| 429 | ⚠️ 记录，不重试 |
| 500+ | ⚠️ 记录，不重试 |

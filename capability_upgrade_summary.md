## 🎛️ 使用说明

### 1. 叶荣添监控
**自动执行**: 每日 9:00/15:00/20:00 自动推送  
**输出格式**:
```
📊 叶荣添监控 (05-24 09:00)
- 最新微博：[时间] [内容摘要]
- 观看量：XX 万
- 更新状态：✅ 有更新 / ⚪ 无更新
- 关联底牌：[方向]
```

### 2. 财经新闻扫描
**自动执行**: 工作日 8:00/18:00 自动推送  
**输出格式**:
```
📰 财经新闻扫描 (05-24 08:00)

🔴 P0 级重大新闻:
- [来源] 新闻标题 (关键词)
- 关联方向：[七张底牌方向]
- 潜在影响：[利好/利空 + 标的]

🟡 P1 级重要新闻:
- [来源] 新闻标题
```

### 3. 市场监控仪表盘
**文件位置**: `strategicimplementation/market_monitoring_dashboard.md`  
**更新频率**: 每日 15:30 自动刷新  
**内容**: 大盘状态、七张底牌监控、持仓表现、异动告警

### 4. 产业链知识图谱
**文件位置**: `portfolio/industry_knowledge_graph.md`  
**覆盖范围**: 七张底牌完整产业链  
**内容**: 上下游关系、核心公司、依赖关系、风险传导

### 5. 多模型路由
**文件位置**: `config/model_routing.md`  
**策略**: 按任务复杂度自动选择最优模型  
**示例**:
- 日常对话 → Qwen3.5-Plus
- 深度分析 → Qwen3.5-Max
- 代码生成 → Claude-Opus
- 图像理解 → GPT-4o

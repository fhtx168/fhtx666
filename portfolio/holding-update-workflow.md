# 持仓更新协作流程（方案 B）

## 触发时间
**每个交易日 16:30 收盘后**

---

## 你的操作（1 分钟）

### 步骤 1：复制持仓
从你的券商 APP/软件复制持仓数据，格式示例：

```
持仓明细 (2026-05-25)
总资产：3338 万
总市值：3474 万
浮动盈亏：+309 万 (+9.83%)

个股持仓：
澜起科技  10 万股  成本 68.5  现价 72.3  盈亏 +5.5%
中际旭创  8 万股   成本 125   现价 131   盈亏 +4.8%
沪电股份  15 万股  成本 28.6  现价 30.1  盈亏 +5.2%
...
```

### 步骤 2：发送给我
直接粘贴到对话框，发送即可。

**或者**：截图也可以，我会 OCR 识别。

---

## 我的操作（30 秒）

收到你的持仓数据后，我自动执行：

### 1. 数据提取
- 解析每只股的代码、数量、成本、现价
- 计算涨跌幅、市值、盈亏贡献

### 2. 异动识别
- 筛选涨跌幅 ≥5% 的股票
- 识别 Top5 盈利 / Top5 亏损

### 3. 报告生成
生成精简版持仓报告：

```markdown
## 📊 持仓日报 (2026-05-25)

### 核心指标
| 指标 | 数值 |
|------|------|
| 总资产 | 3338 万 |
| 总市值 | 3474 万 |
| 浮动盈亏 | +309 万 (+9.83%) |
| 今日盈亏 | +XX 万 (+X.XX%) |

### 🔥 今日异动 (≥5%)
**大涨**：
- 澜起科技 +5.5% (+XX 万)
- 沪电股份 +5.2% (+XX 万)

**大跌**：
- XXX -6.2% (-XX 万)

### 💰 盈利 Top5
1. 澜起科技：+XX 万
2. ...

### ⚠️ 亏损 Top5
1. XXX：-XX 万
2. ...

### 📝 备注
[简要点评]
```

### 4. 归档保存
- 保存到 `portfolio/daily-tracking.md`
- 更新 `portfolio/holdings.md`（如有新数据）
- Git 提交：`feat: 持仓更新 2026-05-25`

- 记录到 `.sync/backup-log.md`

## 一、系统目标

围绕飞鸿的中长线投资策略，构建一个**数据驱动、AI辅助、风险可控**的投研体系，覆盖：
- 每日行情监控与推送
- 持仓个股深度研究
- 财经资讯情绪分析
- 券商研报结构化解析
- 技术指标辅助判断
- 组合风控与预警
- 未来扩展（旅游/康养等生活服务）

### DeepSeek v4-flash
| 职责 | 禁止 |
|------|------|
| 大盘复盘、趋势判断、资金流向解读 | 不接普通新闻 |

## 六、每日任务排期

### 08:30 早间推送
```
【早间速览】
1. 隔夜外围与A股大势简评（1-2句）
2. 今日重点行业/板块情绪与催化
3. 持仓组合风控概览（集中度/回撤/杠杆）
4. 今日关键风险点 + 避雷提示
```
**执行流程**：Tushare拉数据 → DeepSeek分析 → 推送

### 12:30 午间推送
```
【午间小结】
1. 上午大盘强弱 + 主力资金流向（一句话）
2. 热门板块轮动与强弱排序
3. 自选/持仓异动提示（仅异常，无刷屏）
4. 下午观察要点 + 关键支撑/压力
```
**执行流程**：快速查TOP20持仓现价 → DeepSeek一句话总结 → 推送

### 16:00 收盘复盘
```
【收盘复盘】
1. 全天大盘/板块收官与资金流向总结
2. 强势赛道 + 轮动逻辑 + 明日主线观察
3. 持仓全面风控体检（回撤/集中度/盈亏比）
4. 明日关键窗口 + 风险预警 + 重点关注标的
5. **明确投资建议**（买入/持有/减仓建议，含逻辑）
```
**执行流程**：Tushare拉日线 → Python算指标 → DeepSeek复盘+风控 → 推送

### 21:00 晚间推送
```
【晚间分析】
1. 收盘复盘补充与晚间重要公告解读
2. 龙虎榜/资金面变化
3. 次日策略展望 + **明确操作建议**
4. **投资机会**（如有，直接给出标的+逻辑+仓位建议）
```
**执行流程**：拉公告/龙虎榜 → 通义千问解读 → 推送

## 七、持仓股重点关注体系

### 分级管理
| 级别 | 范围 | 监控频率 | 分析深度 |
|------|------|---------|---------|
| **核心（20只）** | 按市值排序的TOP20重仓 | 每日4次推送 | 全面（行情+基本面+技术+情绪） |
| **关注（20只）** | 中等仓位 | 每日涨跌检查 | 涨跌阈值+重大公告 |
| **基础（19只+港股9只）** | 轻仓/港股 | 每日涨跌检查 | 涨跌阈值+重大事件 |

### 异动触发规则（随时推送）
| 触发条件 | 动作 |
|---------|------|
| 个股涨跌幅 ±5%（可调） | 即时推送异动提醒 |
| 突发重大利好/利空舆情 | 即时推送情绪预警 |
| 关键技术位突破/破位 | 即时推送技术信号 |
| 组合回撤/集中度超标 | 即时推送风控预警 |

### 持仓股档案建设
- 为每只持仓股建立独立档案
- 内容：基本面、行业逻辑、技术面、券商评级、历史异动
- 建设顺序：按市值从高到低，分批完成
- 动态更新：重大事件/财报季/异动时更新

### 随时查询能力
用户可随时指令查询任一个股，立即输出：
- 行情 + 基本面 + 情绪 + 风控 + 技术信号
- **明确投资建议**（买入/持有/减仓/回避）

### 投资建议原则
> 大胆明确，基于数据，不模棱两可

| 原则 | 说明 |
|------|------|
| **数据驱动** | 每个建议必须有数据支撑（财务/技术/资金/情绪） |
| **逻辑清晰** | 说明为什么，不单纯给结论 |
| **敢于表态** | 该说买就说买，该说减就说减，不骑墙 |
| **匹配策略** | 符合中长线风格，不推荐短线标的 |
| **风险提示** | 明确给出风险点和止损条件 |

## 📞 应急方案

### 场景 1：C 盘再次告急（<5 GB）

```powershell
# 立即执行
1. 运行 cleanup_disk.ps1（清理临时文件）
2. 清空回收站
3. 检查 Windows Update 缓存
4. 如仍不足，考虑云电脑扩容
```

### 场景 2：Junction 链接失效

```powershell
# 重建 junction
cmd /c mklink /J "C:\Users\Admin\Downloads" "E:\Users\Admin\Downloads"
# 其他目录类似
```

### 场景 3：百度网盘同步失败

```
1. 检查网络连接
2. 重新登录百度网盘
3. 检查同步空间配置
4. 手动上传重要文件
```

### 场景 4：E 盘故障

```
1. 数据仍在云端（百度网盘）
2. 从云端下载到新位置
3. 重建 junction 链接
4. 恢复应用配置
```

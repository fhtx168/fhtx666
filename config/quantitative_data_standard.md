## 📋 量化数据清单（强制执行）

### 每次操盘计划必须包含以下 7 大类量化数据：

#### 1️⃣ 实时行情数据（Tushare/AKShare）

**必须获取**:
- [ ] 现价（元）
- [ ] 市值（亿）
- [ ] 流通市值（亿）
- [ ] PE(TTM)
- [ ] PB
- [ ] 换手率（%）
- [ ] 5 日/10 日/20 日涨跌幅（%）
- [ ] 成交量/成交额

**数据源优先级**:
1. Tushare Pro（主数据源）
2. AKShare（备用数据源）
3. 券商 APP（交叉验证）

**验证方法**:
```python
# 交叉验证
tushare_price = get_price_from_tushare()
akshare_price = get_price_from_akshare()
assert abs(tushare_price - akshare_price) < 0.05  # 偏差<5%
```

---

#### 2️⃣ 业绩量化数据（一致性预期）

**必须获取**:
- [ ] 2026Q1 实际营收（亿）+ 同比（%）
- [ ] 2026Q1 实际净利润（亿）+ 同比（%）
- [ ] 2026Q1 毛利率（%）
- [ ] 2026Q1 ROE（%）
- [ ] 2026 全年预测营收（亿）+ 同比（%）
- [ ] 2026 全年预测净利润（亿）+ 同比（%）
- [ ] 2027 年预测营收（亿）+ 同比（%）
- [ ] 2027 年预测净利润（亿）+ 同比（%）
- [ ] 机构覆盖数量（家）

**数据源**:
- 东方财富 Choice（机构一致性预期）
- 同花顺 iFinD
- 公司财报（巨潮资讯/上交所/深交所）

**验证方法**:
```python
# 财报 vs Tushare 对比
finance_report_data = get_from_report()
tushare_data = get_from_tushare()
assert abs(finance_report_data - tushare_data) < 0.01  # 偏差<1%
```

## 🎯 量化评估标准

### PEG 估值标准

| PEG | 估值评级 | 仓位指引 |
|-----|----------|----------|
| **<0.3** | ⭐⭐⭐⭐⭐ 显著低估 | S++ 级（25%） |
| **0.3-0.5** | ⭐⭐⭐⭐ 低估 | S+ 级（20%） |
| **0.5-0.8** | ⭐⭐⭐ 合理 | A 级（15%） |
| **0.8-1.2** | ⭐⭐ 略高估 | B 级（10%） |
| **>1.2** | ⭐ 高估 | C 级（0% 退出） |

### 技术面评分标准

| 条件 | 得分 |
|------|------|
| 均线多头排列 | +30 分 |
| MACD 金叉 | +20 分 |
| MACD 零轴上方 | +20 分 |
| 股价>MA20 | +15 分 |
| 股价>MA60 | +15 分 |
| **总分** | **100 分** |

**评级**:
- 80-100 分：技术面强势（S 级）
- 60-79 分：技术面中性（A 级）
- <60 分：技术面弱势（B/C 级）

## ⚠️ 数据验证流程

### 交叉验证（必须执行）

```python
# 1. 股价验证
tushare_price = get_price_from_tushare()
akshare_price = get_price_from_akshare()
assert abs(tushare_price - akshare_price) < 0.05, "股价偏差>5%"

# 2. 业绩验证
finance_report = get_from_report()
tushare_finance = get_from_tushare()
assert abs(finance_report - tushare_finance) < 0.01, "业绩偏差>1%"

# 3. 估值验证
tushare_pe = get_pe_from_tushare()
research_pe = get_pe_from_research()
historical_pe = get_historical_pe()
final_pe = median([tushare_pe, research_pe, historical_pe])
```

### 数据质量检查

- [ ] 所有数据来自权威数据源（Tushare/AKShare/Choice/iFinD）
- [ ] 股价数据交叉验证（偏差<5%）
- [ ] 业绩数据交叉验证（偏差<1%）
- [ ] 估值数据多源对比（取中位数）
- [ ] 记录数据获取时间和来源
- [ ] 偏差>5% 时告警并重新获取

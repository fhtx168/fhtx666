## 📋 投资计划必备要件（强制执行）

### 第一部分：量化数据（7 大类）

#### 1️⃣ 实时行情数据
- [ ] 现价、市值、PE、PB
- [ ] 换手率、5 日涨跌幅

#### 2️⃣ 业绩量化数据
- [ ] 2026Q1 实际（营收/净利润/增速）
- [ ] 2026 全年预测（机构一致性）
- [ ] 2027 年预测（机构一致性）

#### 3️⃣ 机构持仓变化
- [ ] 机构持仓占比 + 环比
- [ ] 北向资金占比 + 环比
- [ ] 社保/大基金/华为哈勃

#### 4️⃣ 技术面量化指标
- [ ] 均线系统（MA5/10/20/60/120）
- [ ] MACD（金叉/死叉）
- [ ] 多头排列验证

#### 5️⃣ 资金流向
- [ ] 主力净流入（近 5 日）
- [ ] 北向净流入（近 5 日）
- [ ] 大单/中单/小单占比

#### 6️⃣ 估值量化
- [ ] PEG（2026 年预期）
- [ ] 历史估值分位（5 年）
- [ ] 目标价/目标市值

#### 7️⃣ 风险量化
- [ ] 波动率（20 日/60 日）
- [ ] Beta、最大回撤
- [ ] 止损位（-20% 纪律）

#### 1️⃣ 市场风险

- [ ] 美国管制升级风险
- [ ] 市场情绪转冷风险

#### 2️⃣ 个股风险
- [ ] 估值偏高（历史分位>70%）
- [ ] 波动率高（>5%）
- [ ] 短期涨幅大（>50%）
- [ ] 市值小（<300 亿）
- [ ] 竞争加剧风险

#### 3️⃣ 流动性风险
- [ ] 融资比例（警戒线 50%）
- [ ] 现金储备（目标 10%）
- [ ] 持仓集中度（上限 150%）

#### 4️⃣ 技术面风险
- [ ] 止损触发概率（30 日）
- [ ] 止损位（-10% 警戒/-20% 止损）
- [ ] 应对策略（触及减半/-20% 清仓）

#### 5️⃣ 极端情景压力测试
- [ ] 美股崩盘（-20%）→ 组合回撤/应对
- [ ] A 股系统性风险（-15%）→ 组合回撤/应对
- [ ] 板块回调（-25%）→ 组合回撤/应对
- [ ] 单一标的暴雷（-50%）→ 组合回撤/应对

#### 6️⃣ 风险调整后收益
- [ ] 夏普比率（目标>2.0）
- [ ] 最大回撤（目标<-40%）
- [ ] 年化波动率（目标<35%）
- [ ] Beta（目标<1.5）
- [ ] 预期年化收益（目标>50%）

## 🎯 风险评估标准

### 风险等级划分

| 等级 | 概率 | 影响 | 应对 |
|------|------|------|------|
| **低风险** | <20% | 轻微 | 监控即可 |
| **中风险** | 20-40% | 中等 | 制定应对策略 |
| **高风险** | 40-60% | 严重 | 立即对冲 |
| **极高风险** | >60% | 灾难 | 减仓/清仓 |

### 止损纪律

| 触发条件 | 操作 | 执行时间 |
|----------|------|----------|
| **-10% 警戒位** | 减半仓位 | 当日收盘前 |
| **-20% 止损位** | 清仓 | 当日收盘前 |
| **基本面恶化** | 清仓 | 立即执行 |
| **黑天鹅事件** | 清仓 | 立即执行 |

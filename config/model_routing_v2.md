## ⚠️ 注意事项

### 1. 成本控制
- Qwen-Max 成本约为 Qwen-Plus 的 3-5 倍
- 优化后每日成本增加约 30-50%
- **建议**: 监控月度成本，如超预算适当降级

### 2. 降级策略
**当 Qwen-Max 不可用时**:
- S 级任务 → Qwen-Plus + thinking:on
- A 级任务 → Qwen-Plus + thinking:moderate
- B/C 级任务 → 延迟执行

**当系统负载高时**:
- 优先保障 S 级任务
- A 级任务可延迟 1-2 小时
- B/C 级任务可暂停

### 3. 性能监控
**每周检查**:
- Qwen-Max 任务完成率
- 平均响应时间
- 超时任务数量
- 用户满意度反馈

**每月优化**:
- 根据实际效果调整模型配置
- 优化 timeout 参数
- 调整优先级队列

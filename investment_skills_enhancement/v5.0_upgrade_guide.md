## 🔧 如何使用 v5.0 框架

### 场景 1: 新股评估

```bash
# 步骤 1: 三十五维全面分析
使用 skills/thirty-five-dimensions-scorer/SKILL.md
输出：综合评分（如 85 分）

# 步骤 2: 七张底牌契合度评估
使用 skills/seven-cards-evaluator/SKILL.md
输出：契合度评分（如 90 分）

# 步骤 3: CPU 之战专项评估（新增）
使用 skills/cpu-war-evaluator/SKILL.md（待创建）
输出：CPU 之战评分（如 88 分）

# 步骤 4: 三重过滤
85 分 + 90 分 + 88 分 = S+ 级（超配 20%）

# 步骤 5: 催化剂验证
检查未来 6-12 个月催化剂（P0/P1/P2 级）

# 步骤 6: 生成交易计划
输出建仓价格、目标价、止损价、仓位配置
```

### 场景 2: 持仓复核

```bash
# 步骤 1: 重新评估三十五维评分
对比上次评分，识别变化（±10 分触发告警）

# 步骤 2: 检查七张底牌逻辑
叶荣添最新观点是否支持该方向

# 步骤 3: 检查 CPU 之战评分
ARM 架构授权是否变化
产业链位置是否提升

# 步骤 4: 催化剂是否兑现
P0 级催化剂已兑现 → 考虑止盈
无新催化剂 → 考虑减仓

# 步骤 5: 生成调仓建议
持有/加仓/减仓/清仓
```

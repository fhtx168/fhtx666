## 🔧 备用数据源切换逻辑

```python
# 数据源优先级
1. AKShare（首选，免费，中文数据全）
2. Tushare（备用，需 Token，稳定性好）
3. 本地缓存（网络故障时用昨日数据）
```

### 切换条件

| 场景 | 数据源 | 说明 |
|------|--------|------|
| AKShare 成功 | AKShare | 首选 |
| AKShare 失败 + Tushare 可用 | Tushare | 自动切换 |
| 两者都失败 | 本地缓存 | 用昨日数据 |
| 休市时间 | 本地缓存 | 昨日收盘数据 |

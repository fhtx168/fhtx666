## 🛡️ 冲突解决机制

### 数据不一致处理

**发现不一致** → **立即暂停操作** → **两端沟通** → **统一意见** → **更新数据** → **恢复操作**

**优先级**：
1. **叶荣添观点**：以微信端为准（微信端直接获取）
2. **财务数据**：以桌面端为准（桌面端完整分析）
3. **风险评估**：以桌面端为准（桌面端量化分析）
4. **持仓决策**：两端共同确认（缺一不可）

### 决策分歧处理

**分歧类型**：
- **框架分歧**：暂停操作 → 深入讨论 → 统一框架 → 恢复操作
- **仓位分歧**：取较低仓位 → 补充分析 → 统一意见 → 调整仓位
- **标的分歧**：暂缓建仓 → 对比研究 → 统一意见 → 决定建仓

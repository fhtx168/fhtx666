## C 盘彻底清理方案（目标：25+ GB）

### 可迁移目录清单

| 目录 | 大小 | 迁移难度 | 优先级 | 说明 |
|------|------|----------|--------|------|
| `C:\Program Files (x86)` | 12.19 GB | ⭐⭐⭐ | P1 | 32 位软件，可迁移部分 |
| `C:\Program Files` | 9.05 GB | ⭐⭐⭐ | P1 | 64 位软件，可迁移部分 |
| `C:\Users\Admin\OneDrive` | 未知 | ⭐⭐ | P2 | 云同步文件夹 |
| `C:\Users\Admin\WPSDrive` | 未知 | ⭐⭐ | P2 | WPS 云盘 |
| `C:\Windows\Logs` | 0.04 GB | ⭐ | P3 | 系统日志（可清理） |
| `C:\inetpub` | ~0 GB | ⭐ | P3 | IIS 网站（如不用可删除） |
| `C:\msys64` | 0.31 GB | ⭐⭐ | P3 | Git Bash |

### 迁移策略

#### P1 - 软件迁移（回收 15-20 GB）

**不推荐直接迁移 Program Files**（容易破坏软件），建议：

1. **卸载不用的软件**（优先）
2. **重新安装到 E 盘**（新软件）
3. **使用 junction 迁移特定软件**（如绿色软件）

**可安全迁移的软件**：
- 360 安全卫士 → 卸载或重装到 E 盘
- 办公软件（WPS/Office）→ 部分组件可迁移
- 开发工具（msys64）→ 可迁移

#### P2 - 云同步文件夹（回收 2-5 GB）

**OneDrive**：
```powershell
# 1. 停止 OneDrive 同步
taskkill /f /im OneDrive.exe

# 2. 移动到 E 盘
Move-Item "C:\Users\Admin\OneDrive" "E:\CloudSync\OneDrive" -Force

# 3. 创建 junction
cmd /c mklink /J "C:\Users\Admin\OneDrive" "E:\CloudSync\OneDrive"

# 4. 重启 OneDrive
Start-Process "C:\Users\Admin\AppData\Local\Microsoft\OneDrive\OneDrive.exe"
```

**WPSDrive**：
```powershell
# 类似操作
Move-Item "C:\Users\Admin\WPSDrive" "E:\CloudSync\WPSDrive" -Force
cmd /c mklink /J "C:\Users\Admin\WPSDrive" "E:\CloudSync\WPSDrive"
```

#### P3 - 系统清理（回收 1-2 GB）

**清理 Windows 日志**：
```powershell
# 清理旧日志
Remove-Item "C:\Windows\Logs\*.etl" -Force -ErrorAction SilentlyContinue
Remove-Item "C:\Windows\Logs\CBS\*.log" -Force -ErrorAction SilentlyContinue
```

**禁用 IIS**（如不用）：
```powershell
# 停止 IIS 服务
Stop-Service W3SVC -Force
# 可选：禁用 IIS 功能
```

## 百度网盘同步配置

### 方案 A：百度网盘同步盘（推荐）

**优点**：
- 自动同步，无需手动操作
- 双向同步（本地↔云端）
- 版本历史，可恢复旧版本

**配置步骤**：
1. 安装百度网盘客户端
2. 开通同步空间（会员权益）
3. 选择同步文件夹：
   - `E:\Users\Admin\Documents`（文档）
   - `E:\Users\Admin\Desktop`（桌面）
   - `E:\opcclawai\project`（工作项目）
   - `E:\CloudSync\Backup`（备份数据）

### 方案 B：百度网盘备份（增量）

**优点**：
- 不占用本地空间
- 适合大文件/历史数据

**配置步骤**：
1. 创建备份清单
2. 每周/每月手动备份一次
3. 压缩加密后上传

### 方案 C：混合方案（最佳实践）

```
E:\
├── CloudSync\              # 百度网盘同步
│   ├── Documents\          # 文档（实时同步）
│   ├── Projects\           # 项目代码（实时同步）
│   └── Backup\             # 备份数据（每周更新）
│
├── Users\Admin\            # 用户目录（junction 链接）
│   ├── Downloads -> E:\Users\Admin\Downloads
│   ├── Documents -> E:\CloudSync\Documents
│   └── Desktop -> E:\CloudSync\Desktop
│
└── AppCache\               # 应用缓存
    ├── Google\Chrome\
    ├── Tencent\
    └── Kingsoft\
```

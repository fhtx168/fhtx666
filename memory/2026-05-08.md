# 2026-05-08 防失忆三层机制建立

## 三层记忆架构
### 🥇 每日日志 (`memory/YYYY-MM-DD.md`)
- **作用**：当天所有工作记录，两端启动时自动读取
- **内容**：详细的操作记录、讨论内容、决策过程

### 🥈 同步桥接 (`.sync/channel-sync.md`)  
- **作用**：产出摘要+当前状态，轻量快速检索
- **内容**：关键操作摘要、文件更新记录、状态变更

### 🥇 长期记忆 (`MEMORY.md`)
- **作用**：主线/持仓/配置/理念等持久化状态
- **内容**：投资理念、持仓概况、API密钥、技能管理规则

## 工作流程
1. **每次在任一通道做完重要操作** → 写文件 + 更新 sync 日志
2. **另一端的「我」启动时** → 先读 sync 日志 → 再读每日记录 → 自动追上进度  
3. **重大变更** → sessions_send 推送到另一端即时通知

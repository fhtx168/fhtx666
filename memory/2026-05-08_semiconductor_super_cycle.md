# 国产半导体"超级周期"核心观点记录
## 视频学习摘要（2026年5月8日）

### 一、全球半导体周期与国产反应滞后
- **全球周期**：已持续三年，分两个阶段
  - 2023年：ChatGPT发布后美股算力芯片上涨
  - 2025年：存储芯片涨价阶段
- **当前状态**：美国大空头开始做空美股芯片股
- **国产滞后原因**：产能不足 + 生态不完善，算力卡市场认可度低

### 二、DeepSeek V4与昇腾合作突破
- **合作成果**：联手推出DeepSeek V4
- **技术瓶颈**：Pro版本受限于高端算力，数据吞吐量有限
- **关键突破**：下半年昇腾950超节点批量上市
- **成本优势**：Pro价格将大幅下调，打通企业采购需求逻辑

### 三、超节点技术革命
- **技术原理**：通过先进互联技术将成百上千张卡组成计算单元
- **性能突破**：
  - 解决单卡性能不足问题
  - 突破通讯瓶颈与等待损耗  
  - 实现"1+1>2"的集群效应
- **具体对比**：
  - 华为910C单卡性能 = 英伟达B200的1/3
  - 384超节点性能 = 反超英伟达NVL72达1.7倍
  - 950超节点（8192张卡）对标NVL144，有显著优势
- **三大价值**：性能补位、成本优势、产能验证

### 四、2026年市场格局预测
- **国产占有率**：2026年新增算力市场超50%，反超英伟达
- **华为昇腾份额**：占国产半壁江山
- **出货量预测**：
  - 昇腾卡整体：90万—110万张
  - 950型号：75万—80万张
- **关键转折**：供需卡点突破，生态跑通，产能成关键
- **周期判断**：超级周期逻辑正式形成

### 五、产业机遇与受益方向
- **需求特征**：超节点对芯片卡数量需求指数级增长
- **订单爆发**：带动产业链增量订单爆发
- **第一波受益主体**：
  1. 华为昇腾及超节点硬件供应商
  2. 国产算力生态配套企业  
  3. 基于国产算力的AI大模型与应用开发商

### 六、全球格局影响
- **国内影响**：推动国内半导体进入"超级周期"
- **国际挑战**：改变全球算力市场格局，挑战美股芯片股长优势
- **生态演进**：加速全球算力生态多元化
